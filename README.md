# My Portfolio Workspace

Repo da duoc tach theo huong de mo rong:

- `frontend/`: Astro frontend app (toan bo giao dien va content hien tai)
- `backend/`: de trong de bo sung sau (neu can API, auth, worker...)

## Chay nhanh tu root

```bash
npm run install:frontend
npm run dev
```

## Scripts o root

- `npm run install:frontend`: cai dependencies cho frontend
- `npm run dev`: chay Astro dev server
- `npm run build`: build frontend
- `npm run preview`: preview build frontend

## Lam viec truc tiep voi frontend

Neu muon thao tac truc tiep trong app:

```bash
cd frontend
npm install
npm run dev
```
